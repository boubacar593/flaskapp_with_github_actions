# Flask CI/CD App

Ce projet est une application web simple développée avec **Flask** et déployée automatiquement grâce à **GitHub Actions** dans un pipeline de **CI/CD**.

## Objectif du projet

L’objectif est de démontrer mes compétences en :

- Développement d’une application web avec Flask
- Gestion du code avec Git et GitHub
- Mise en place d’un pipeline CI/CD avec GitHub Actions
- Déploiement automatique de l’application

## Technologies utilisées

- Python
- Flask
- GitHub Actions
- GitHub

## Fonctionnement du CI/CD

À chaque `push` sur la branche principale :

- le code est récupéré automatiquement,
- les dépendances sont installées,
- l’application est testée,
- puis le processus de déploiement est lancé via GitHub Actions.
